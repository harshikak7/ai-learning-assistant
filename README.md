# AI Learning Assistant (MERN Stack Project)

This is a full-stack AI learning assistant project built using the MERN stack.
Currently, the backend authentication system and core database models are implemented.

## Tech Stack

Backend:

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt

Frontend:

* React (setup started)

## Features Completed So Far

Authentication Module:

* User registration
* Login system
* Password hashing
* JWT token authentication
* Protected routes middleware
* Update profile
* Change password

Database Models Created:

* User
* Document
* Flashcard
* Quiz
* Chat History
